# 주제 선정 :
의류 쇼핑몰 - 겨울철 의상. 종류는 상의, 하의, 아우터, 신발, 액세서리. 
'로그인'하여 '제품'을 '장바구니'에 담고 '구매'를 진행할 수 있으며
'구매내역'을 확인 가능하고, 구매한 제품의 '리뷰작성'을 할 수 있는 사이트입니다.
